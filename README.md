# Cookbook 

## Описание
Проект представляет собой кулинарную книгу с различными категориями рецептов.

## Структура
- **супы.md**: рецепты супов.
- **салаты.md**: рецепты салатов.
- **десерты.md**: рецепты десертов.

## Использование
- Клонируйте репозиторий.
- Редактируйте рецепты или добавляйте новые.
- Управляйте изменениями с помощью Git.
